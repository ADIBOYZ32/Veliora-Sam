# Veliora-Sam

A React + Vite + Three.js project.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/ADIBOYZ32/Veliora-Sam.git
cd Veliora-Sam
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
